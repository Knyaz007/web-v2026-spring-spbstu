Задача 5. Реализовать механику списка в книжном интернет-магазине
Создать компонент списка в виде дерева (tree). Добавить возможность
скрывать/раскрывать ветки.
Дизайн дерева остается за студентом.
Структура предполагаемого дерева (по уровням):
- Автор;
 - Издательство;
 - Жанр;
 - Название книги.

Эта реализация включает:

Иерархическое дерево с уровнями: Автор → Издательство → Жанр → Название книги

Возможность сворачивать/разворачивать ветки кликом по кнопкам ▼/▶

Стилизованный дизайн с разными цветами для каждого уровня

Статистика (количество авторов, издательств, жанров и книг)

Кнопки управления: "Развернуть всё", "Свернуть всё", "Сбросить"

Поиск книг, авторов и жанров

Адаптивный дизайн для мобильных устройств

Анимации при наведении и сворачивании
